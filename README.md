# Cpp.Include.Linux
These C++ include files I use to simplify some C++ tasks on Linux.  
License: **GPL-3.0-or-later**

## itsoftware-linux-core.h
Has the following types:
    * ItsTimer
    * ItsGuid
    * ItsError
    * ItsDirectory
    * unique_so_handle
    * unique_file_handle
    * ItsFile
    * ItsPath
    * ItsFileMonitor
    * ItsDaemon

    * ItsSocketStreamClient

    * ItsSocketDatagramClient
    * ItsPipe
    * ItSvMessageQueue
    * ItsFifoServer
    * ItsFifoClient
